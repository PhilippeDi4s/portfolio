Cardápio de uma loja

Aplicação web com lista de produtos e carrinho de compras funcional, utilizando dados carregados a partir de um arquivo JSON para renderização dinâmica dos produtos. O sistema permite adicionar, remover e ajustar quantidades de itens, exibe confirmação de pedido via modal e possibilita iniciar um novo pedido. O layout é totalmente responsivo e conta com estados de hover e foco, garantindo uma experiência de uso clara e intuitiva.

Gerador de Ticket para Conferência

Aplicação web para geração de tickets de conferência a partir do preenchimento de um formulário. O sistema permite que o usuário insira seus dados pessoais e faça o upload de uma imagem de avatar, que é exibida diretamente no ticket gerado após o envio bem-sucedido do formulário.

O projeto conta com validações completas, garantindo o preenchimento correto dos campos, verificação do formato de e-mail e restrições de tamanho e tipo de arquivo para o upload da imagem. A interface foi desenvolvida com foco em acessibilidade, permitindo navegação total via teclado e suporte a leitores de tela, além de layout totalmente responsivo e estados visuais de hover e foco para melhor experiência do usuário.

Dashboard Interativo de Análise Salarial na Área de Dados

Dashboard interativo para análise de salários na área de dados, construído a partir do tratamento e visualização de um DataFrame com informações do mercado. A aplicação apresenta métricas gerais como salário médio, salário máximo, total de registros e cargos mais frequentes, oferecendo uma visão clara e objetiva dos dados analisados.

O painel permite filtros dinâmicos por ano, senioridade, tipo de contrato e tamanho da empresa, além de visualizações interativas que incluem ranking de cargos por salário médio, distribuição salarial, proporção dos tipos de trabalho e mapa de salários médios de Cientistas de Dados por país. O projeto entrega uma análise exploratória completa com foco em usabilidade, clareza visual e apoio à tomada de decisão.
